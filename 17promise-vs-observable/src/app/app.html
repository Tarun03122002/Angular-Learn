Diff B/w Promise and obs

Promise is native to JS whereas Observable is not native to JS .It is a part of third party native library called RxJS
Promise is eager (As soon as we create it,it will return some values) whereas observable is lazy(Emit values only if we subscribe)
Returns single value whereas observable emits single or multiple values
Provide methods for success and error using then and catch whereas Observable provides method for success,errorand completion using next(),error(),complete()
Promises handles only asynchronous data whereas observable handles both synchronous and asynchronous data
